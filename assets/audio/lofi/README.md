# Lo-Fi Music Tracks

Place your royalty-free MP3 tracks in this folder.

The application will **automatically discover and play** any `.mp3`, `.ogg`, or `.wav` files here.

## Where to Download Tracks

Visit: https://pixabay.com/music/

Search for:
- LoFi
- Chill
- Study
- Ambient
- Relaxing
- Chillhop

Download in **MP3 format** and place directly in this folder.

## Example

```
assets/audio/lofi/
├── lofi-chill-01.mp3
├── midnight-rain.mp3
├── coffee-shop-beats.mp3
└── study-session.mp3
```

## Future Playlists

You can add additional mood folders alongside this one:

```
assets/audio/
├── lofi/        ← current
├── rain/        ← future
├── fantasy/     ← future
└── cyberpunk/   ← future
```

Each sub-folder will appear as a separate playlist in the music player settings.
